Import aus Dump File

In MySQL Workbench kann man Daten aus einem Dump File importieren.

Import von Daten

  1. Zu Navigator | Administrator gehen

  2. Dort Data Import/Restore auswählen

  3. In den Importoptionen die zweite Auswahl treffen und das Dump-File auswählen

  4. Import starten
