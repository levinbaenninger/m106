Export in Dump File

Um in MySQL Workbench Daten zu exportieren, können wir ein Dump File erstellen.

Export von Daten

  1. Zu Navigator | Administrator gehen

  2. Dort Data Export auswählen

  3. Das zu exportierende **Schema und Tabellen auswählen**

  4. In den Exportoptionen die zweite Auswahl treffen

  5. Sowohl Create Dump in a Single Transaction, als auch Include Create Schema auswählen

  6. Export starten
